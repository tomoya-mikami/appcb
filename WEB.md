# ウェブアプリケーション仕様書

## アプリケーションを使用する準備
ウェブアプリケーションはプロジェクト単位で使用します.プロジェクトの登録には以下の情報が必要です

|name               |description                                                          |
|-------------------|---------------------------------------------------------------------|
|プロジェクト名      |プロジェクトで使用する名前. アルファベットで指定する                      |
|Google Map Api Key |アプリケーションではGoogle Map Apiを使用します.そのため、プロジェクトごとにApi keyが必要になります |

## アプリケーションの機能

アプリケーションでは、位置情報の収集・表示を行います. 収集では事前に登録されたアイコンでの情報収集, 画像アップロードによる情報収集を行うことができます.

### アイコンによる位置情報の収集

url : https://mind-cpndd.slis.tsukuba.ac.jp/projects/プロジェクト名/map

特定の場所で何があったかを事前に登録したアイコンを用いて報告するページです

### 画像による位置情報の収集

url : https://mind-cpndd.slis.tsukuba.ac.jp/projects/プロジェクト名/image

特定の場所で何があったかを画像アップロードを用いて報告するページです

### 画像による位置情報の収集

url : https://mind-cpndd.slis.tsukuba.ac.jp/projects/プロジェクト名/analytics

収集したデータを表示するページです. 3秒おきにデータを更新します.
